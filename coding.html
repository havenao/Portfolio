CODING PROJECTS

I love making fun and useful programs, check out what I've made below!

Dominion Initializer

An app that assists in choosing which cards and extras to use for a game of Dominion. Dominion is a deck-building card game with over 400 unique cards. In each game you must select a small set of those cards to use, so this app makes that process easier!

FizzBuzzVizzualizer

This is a silly little app that displays the game FizzBuzz visually. In this version you may choose values for Fizz and Buzz, or automatically change the values, creating generative art.

Pennsylvania Bones v1

In this game, you play as an archeologist trapped in a burning museum. Try to save as many artifacts as you can and collect special items to aid you as you descend to escape. I made this using Ruby and the Gosu library.

Pennsylvania Bones v2

I am remaking the game using Unity. I have added animations and some UI elements. I have put the game on a temporary pause to focus on other areas of programming. You can play the unfinished current version on my itch.io page
(I recommend playing in full screen mode).
